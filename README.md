## How to Run

### Prerequisites
- Node.js installed on your system

### Installation & Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/MONESHMergu/portfolio.git
   cd portfolio
   ```

2. Start the development server:
   ```bash
   node server.js
   ```
   Or use npm:
   ```bash
   npm start
   ```

3. Open your browser and navigate to:
   ```
   http://localhost:3000
   ```

## Customization
To customize this portfolio:

1. **Update Personal Information**: Edit the `resumeData` object in `main.js`
   - About section (name, title, description, skills)
   - Experience entries
   - Project details and links
   - Achievements

2. **Modify Styling**: Adjust colors and themes in `styles.css`
   - Color schemes (primary: #1e90ff, accent: #00bfff)
   - Background gradients
   - Animation timings

3. **Update Links**:
   - Social media links in `index.html`
   - Project repository URLs in `main.js`
