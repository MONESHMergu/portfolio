Setup Instructions

  1. Install Node.js from nodejs.org
  2. Open a terminal/command prompt in the portfolio folder
  3. Run the command: node server.js
  4. Open your browser and navigate to http://localhost:3000
Alternative: If you don't want to install Node.js, you can use any web server software to serve these files, or upload them to a web hosting service.
